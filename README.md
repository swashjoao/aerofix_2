# Configuração e Execução do Projeto

Este guia demonstra como configurar e executar o **Backend** e o **Frontend** deste projeto.

## Como Iniciar o Projeto

Siga os comandos abaixo, simulando uma sessão de terminal:

### 1. Inicialização do Backend

```bash
$ cd backend

backend $ npm install

backend $ npm run dev
```

### 2. Inicialização do Frontend

```
O Frontend pode ser acessado abrindo frontend/index.html
```
